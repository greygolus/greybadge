How live sync works

Exact mode refreshes when the displayed minute changes. Low-wear mode refreshes every five minutes. Animations run on the badge between syncs; keep this tab open and use the badge button if your model returns to its charging screen.

“Until disconnected” has no timer cap and asks the browser to keep the screen awake. Starting Clock Mode temporarily replaces the playlist on the physical badge; your project here stays untouched.
